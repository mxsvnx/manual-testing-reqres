# Тестирование формы авторизации — Reqres.in

Учебный проект по ручному тестированию. Объект тестирования — форма входа и регистрации публичного тестового сайта [Reqres.in](https://reqres.in).

## Что сделано

- Чек-лист из 32 проверок
- 15 тест-кейсов (позитивные, негативные, граничные значения)
- 3 баг-репорта с шагами воспроизведения и приоритетом

## Техники тест-дизайна

- Эквивалентное разбиение (Equivalence Partitioning)
- Анализ граничных значений (Boundary Value Analysis)
- Таблица принятия решений (Decision Table)

## Структура репозитория

```
project1-manual-testing/
├── checklist.md       # Чек-лист проверок
├── test-cases.md      # Тест-кейсы
└── bug-reports.md     # Баг-репорты
```

## Инструменты

- Ручное тестирование в браузере (Chrome, Firefox)
- Markdown для документации
